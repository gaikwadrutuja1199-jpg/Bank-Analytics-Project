# Bank Analytics Project

## Tools Used
- SQL (Data extraction, joins, KPI calculations)
- Excel (Data cleaning, pivot tables, analysis)
- Power BI (Interactive dashboards)
- Tableau (Data visualization and insights)

---

## Objective
To analyze banking data in order to understand customer behavior, loan performance, credit risk, and overall business performance, enabling data-driven decision-making.

---

## Dataset Overview
The dataset includes customer information, account details, loan records, debit and credit transactions, and default status.  
Data was cleaned and transformed before analysis.

---

## Key KPIs
- Total Customers  
- Total Loan Amount  
- Default Rate  
- Average Account Balance  
- Debit vs Credit Analysis  
- Customer Segmentation  

---

## SQL Analysis
- Data cleaning and validation  
- Joining multiple tables  
- Aggregate functions for KPI calculation  
- CASE statements for customer segmentation
